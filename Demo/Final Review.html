By popular demand ... data files for an additional SQL exercise.

Note: This exercise uses only INNER and LEFT joins; but not RIGHT or OUTER joins. Therefore you may use SQLite if you wish or MySQL as you prefer (taking care to distinguish the data types appropriately.)

  • Museum of Modern Art - artists (as CSV file)
  • Museum of Modern Art - works of art (as CSV file)
  • Museum of Modern Art - artist nationalities that are on file
  • SOURCE: https://github.com/MuseumofModernArt/collection
  • Modifications: For the purpose of this study, only artists with a known biography are included; only works which have been catalogued are included; and only works with a corresponding artist are included (read: referential integrity is enforced).
  • Sample Exercise questions
    1. How many French artists are represented in MoMA's collection?
    2. How many works of art by French artists are represented in MoMA's collection?
    3. Display the Medium and number of artworks by Iranian artists in alphabetical order by Medium.
    4. Display each Medium that includes 'Pencil' in combination with 'Paper'. (Hint: consider the usage of the keyword DISTINCT.)
    5. List the title of the artwork, the artist's display name and the date acquired for all artworks acquired in 1929.
    6. The how many works of art the museum acquired every year in chronological order by year.
    7. Does this database comply with referential integrity? How do you know? Which query or queries did you run to answer this question?
    8. Come up with questions to ask of your own!

Along with data for a NoSQL / MongoDB exercise:

  • Import the U.S. 2010 school districts file as a JSON file ... source: https://www.census.gov/geo/maps-data/data/gazetteer2010.html and converted to JSON format

    • mongoimport --db your_database_name --collection sd --host class-mongodb.cims.nyu.edu --username your_username --password your_password --type json --file sd.json --jsonArray

  • Sample exercise questions:
    1. Which five school districts have the highest populations in 2010? Display the district name, state and population only.
    2. Which five school districts in New York State have the highest populations in 2010? Display the district name and population only.
    3. List all of the school districts in Vermont with a size of over 20 square miles.
    4. Are there any school districts that do not offer either kindergarten or pre-kindergarten? (Such a school district would start in 1st grade.)
    5. What are the possible values for the highest grade? (Hint: Use the keyword DISTINCT.)
    6. Come up with questions to ask of your own!
